Detailed Description

Сравнение различных хэш-функций для строк.

Сравниваются 4 хэш-функции: POLY (наша текущая), RS, JS, DJB. ROT13 исключён из-за аномально высокого количества коллизий (~15000), что делает его непригодным для хэш-таблиц.
