◆ insert()

void HashTable::insert ( const Flight & f)

Вставка объекта в таблицу.

Parameters
fОбъект Flight.

Если ключ (номер рейса) уже существует в соответствующей корзине, объект добавляется в существующий вектор. Если ключ отсутствует, создаётся новая запись. Коллизия увеличивается, если при вставке нового уникального ключа корзина уже была непуста.
