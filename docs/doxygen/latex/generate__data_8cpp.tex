\doxysubsection{Detailed Description}
Генератор тестовых данных для лабораторной работы №2 (поиск данных). 

Создаёт CSV-\/файлы с информацией об авиарейсах. Номера рейсов (ключи поиска) генерируются из ограниченного набора, что обеспечивает многократные повторения для тестирования поиска всех вхождений.\hypertarget{generate__data_8cpp_usage}{}\doxysubsection{\texorpdfstring{Запуск}{Запуск}}\label{generate__data_8cpp_usage}
