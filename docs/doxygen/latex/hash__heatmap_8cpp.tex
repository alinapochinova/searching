\doxysubsection{Detailed Description}
Анализ распределения элементов по корзинам хэш-\/таблицы. 

Строит хэш-\/таблицу на 500000 элементах (2000 уникальных ключей), анализирует распределение элементов по корзинам, выводит статистику.
