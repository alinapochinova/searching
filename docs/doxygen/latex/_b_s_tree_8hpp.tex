\doxysubsection{Detailed Description}
Заголовочный файл бинарного дерева поиска (BST). 

Реализует бинарное дерево поиска для хранения объектов \doxylink{class_flight}{Flight}. Ключом является номер рейса (flight\+Number). Поддерживаются дубликаты:\+ в каждом узле хранится вектор всех объектов с одинаковым ключом.
