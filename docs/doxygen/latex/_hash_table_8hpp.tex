Заголовочный файл хэш-\/таблицы с методом цепочек. 

Реализует хэш-\/таблицу, где каждая корзина является unordered\+\_\+map$<$key, vector$<$\+Flight$>$$>$. Это позволяет хранить дубликаты и корректно подсчитывать коллизии (только для разных ключей, попавших в одну корзину).
