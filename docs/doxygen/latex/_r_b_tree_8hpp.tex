\doxysubsection{Detailed Description}
Заголовочный файл красно-\/чёрного дерева (Red-\/\+Black Tree). 

Реализует самобалансирующееся двоичное дерево поиска с поддержкой дубликатов. Каждый узел хранит ключ (номер рейса) и вектор объектов \doxylink{class_flight}{Flight} с этим ключом.
